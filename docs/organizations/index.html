    Organizations

    This is an incomplete directory of organizations related to the Civil Rights movement in Baltimore. This directory is a working draft and may contain errors. We initially compiled this directory using Google Sheets but migrated the data to Airtable in June 2017.

    Please help us make this directory better! Helpful comments include suggesting organizations that we are missing or correcting any inaccurate information. You can also contact us through our website or send us an email at info@baltimoreheritage.org.
